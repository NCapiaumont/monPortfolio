Nouvelle résolution Linux

Un problème de résolution sous linux ?

Votre résolution ne correspond pas à votre écran ?

Ajouter une résolution

Lancer la commande cvt Hauteur Largeur. Exemple pour la résolution 1920x1200:cvt 1920 1200

On crée ce nouveau mode et on ajoute à la liste des résolutions connues cette nouvelle config grâce à la commande : xrandr –newmode et le modeline précédemment récupéré, puis on l'ajoute à notre écran via xrandr –output –mode .

Exemple pour la résolution 1920x1200 :

  • xrandr --newmode "1920x1200_60" 106.50 1920 1528 1672 1904 1200 903 909 934 -hsync +vsync

Ici, le nom du mode est 1920x1200_60

Ici, le nom de notre sortie est VGA1. Vous pouvez récupérer ce paramètre avec la commande xrandr | grep connected.

Il reste maintenant à appliquer cette résolution : xrandr --output --mode

  • xrandr --output VGA1 --mode 1920x1200_60

executer le script au démarrage de linux

  • $ /etc/gdm3/Init/Default

Récapitulatif

# Crée un nouveau mode pour la résolution

  • xrandr --newmode "1920x1200_60" 106.50 1920 1528 1672 1904 1200 903 909 934 -hsync +vsync

# Ajoute le mode

  • xrandr --addmode VGA1 1920x1200_60

# Applique le nouveau mode

  • xrandr --output VGA1 --mode 1920x1200_60

Skype sur Debian 7.0 "Wheezy"

Installer skype

Skype est disponible pour les architectures amd64 et i386 architectures de Debian 5 et plus récent. En général, on trouve les derniers paquets sur le site de Skype.

Attention, il n'y a pas de dépôt qui possède la dernière version (4.x). Vous devrez donc télécharger le paquet chaque fois qu'il y a une mise à jour. Et comme apt-get ne peut pas vous avertir des mises à jour, on vous conseille d'activer "[x] Vérifier les mises à jour au démarrage" dans l'onglet ‘Avancé’ du menu options de Skype.

i386

Le paquet i386 fonctionne bien avec un système Debian 7.0 i386 standard. Pour l'installer, exécutez les commandes :

  • # wget -O skype-install.deb http://www.skype.com/go/getskype-linux-deb
  • # dpkg -i skype-install.deb

On peut utiliser la commande :

  • # apt-get -f install

pour corriger les problèmes de dépendances manquantes.

Multi-Arch :

A partir de Wheezy, Debian prend en charge Multi-Arch. Les paquets Skype pour i386 fonctionnent bien avec Multi-Arch, vous pouvez avoir des problèmes pour installer les paquets deb 64bits parce qu'ils dépendent des paquets ia32 qui ne sont plus disponibles dans testing et sid à partir de wheezy.

D'abord, il faut activer Multi-Arch :

  • # dpkg --add-architecture i386
  • # apt-get update

Puis, téléchargez le paquet i386 comme on l'a vu plus haut et installez-le :

  • # wget -O skype-install.deb http://www.skype.com/go/getskype-linux-deb
  • # dpkg -i skype-install.deb

Si vous faites plutôt

  • # apt-get -f install

Un certain nombres de paquets i386 seront installés.

Si apt-get n'arrive pas à installer libqtwebkit4:i386 comme dépendance, essayez avec aptitude (si vous savez ce que vous faites)

Les problèmes de son

Si vous avez des problèmes de son, c'est peut-être dû à l'absence de libpulse. On peut l'installer ainsi :

  • # apt-get install libpulse0:i386
